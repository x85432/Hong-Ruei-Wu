興趣

排球

自國中起，閒暇時我最喜歡做的事情就是去打排球。
大二時擔任應數系排球隊的副隊長，大三轉為隊長。

唱歌

由於父親從小就教我們唱歌，讓我練就一身好歌喉。

    代表曲目:
  • 任賢齊 《傷心太平洋》
  • 張宇 《雨一直下》
  • 王傑 《她的背影》
  • 陳奕迅 《愛情轉移》
因為太多老歌了，我也開始學習唱新歌。
目前理想混蛋 《不是因為天氣晴朗才愛你》我能詮釋得比較好！

學習歷程

自古英數不兩立

出生民國93年3月，長於台中市霧峰區。

小學

進入我家旁邊的五福國小就讀，只有一班17人，在國小三年級之前沒有得過班上的第一名。

直到立志未來想當工程師以及某一次英文段考只得了48分，於是問媽媽能不能讓我去補英文，因此英文成為我求學歷程最好的科目沒有之一。

國高中

而在國中時期發現我對於數學的領悟並沒有那麼好，希望能複製英文的成功經驗，於是開始補習數學。
很可惜不是事事如願，了解到數學為工程師的先要條件，令我非常焦慮，直到分科測驗(類似指考)結束，依舊沒有改善
(學測數學前標、分科數學均標)。

大學

於是毅然決然踏入交大應數，希望能彌補自己唯一的短板。
周周轉轉，在大三順利進入交大資工系就讀，並且雙主修應數系，希望能在未來嶄露頭角。

學習歷程

Programming

啟蒙運動

國小國中Minecraft是大家口中炙熱的火球，我也為之著迷。

因為想要從根部成為真正的造物主，Java，成為我的Coding起跑線。
雖然那時大部分的程式碼是public class static void、sysytem.println()...

文藝復興

國中會考結束之後，Python紅透天，讓我意識到寫這個東西好像真的很賺錢，我開始跑到Youtube學習Python， 甚至我還記得那時的播放列表 ，以及授課老師回覆我的留言。

巴洛克時期

高中發現有能力競賽，參加並且使用Python得到佳作，進入培訓後老師讓我開始學習C/C++，一個很討厭的程式語言，什麼都要宣告清清楚楚。
然而我卻開始自己去Youtube找影片學習，並且到Online Judge上測試自己的能力。 大學的計概、轉系考，迫使我將Python與C++的能力提升到一個新的高度。

未來展望

展翅翱翔

  • 莫忘初衷，自國小的工程師之夢，我為自己舖足了一條紅磚大道。
  • 接受改變，儘管我們要記得初衷，但不要侷限自己一定要走哪一條道路，要堅信自己都能夠做的很好。
  • 勇往直前，面對陌生人、陌生領域，只要自己肯開口就有機會，放大機會、抓住機會，我想這是我非常需要的能力。
